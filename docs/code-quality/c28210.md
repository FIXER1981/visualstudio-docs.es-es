# <a name="c28210"></a>C28220
ADVERTENCIA 28210: Las anotaciones del contexto `_On_failure_` no deben estar en un contexto previo explícito

 Las anotaciones `_On_failure_` deben indicarse explícita o implícitamente en el contexto de `__post`, es decir, que se aplicarán después de que la función devuelva un valor.  Use `_drv_out` para asegurarse de ello.
