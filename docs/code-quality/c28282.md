# <a name="c28282"></a>C28282
C28282 de ADVERTENCIA: Las cadenas de formato deben estar en las condiciones previas

 Esta advertencia indica que se encuentra una anotación `__drv_formatString`, que no está en una anotación `_Pre_` (`__drv_in`) (los parámetros de función son de forma predeterminada `_Pre_`). Compruebe si la anotación se ha usado en un bloque explícito con una anotación `_Post_` (`__drv_out`). Si es así, quite la anotación de cualquier bloque de inclusión que la haya colocado en un contexto `_Post_`.
