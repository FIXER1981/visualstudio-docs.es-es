# <a name="ijsdebugproperty-interface"></a>IJsDebugProperty (Interfaz)
Representa un explorador de propiedades.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp
IJsDebugProperty : public IUnknown;  
```
