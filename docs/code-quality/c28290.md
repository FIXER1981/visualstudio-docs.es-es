# <a name="c28290"></a>C28290
ADVERTENCIA C28290: la anotación de la función contiene más externos que el número real de parámetros

 La herramienta de análisis de código notifica esta advertencia cuando la anotación de la función contiene más externos que el número real de parámetros.
