# <a name="c28300"></a>C28300
ADVERTENCIA C28300: < parameter_name >: Operandos de expresión de tipos no compatibles para el operador < operator_name >

 Esta advertencia se indica cuando los tipos de operando de un parámetro no son compatibles con el operador.
