# <a name="extending-properties-and-the-property-window"></a>Ampliación de propiedades y la ventana de propiedades
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La ventana **propiedades** se utiliza para ver y editar las propiedades de tiempo de diseño y los eventos de los objetos seleccionados.  
  
 Al seleccionar un objeto en [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] , las propiedades públicas de ese objeto aparecen en la ventana **propiedades** . Para obtener más información, vea [información general sobre la ventana Propiedades](../extensibility/internals/properties-window-overview.md).
