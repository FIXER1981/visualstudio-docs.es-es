# <a name="c6289"></a>C6289
C6289 de ADVERTENCIA: Operador incorrecto: la exclusión mutua &#124; &#124; sobre siempre es una constante distinta de cero. ¿Desea usar & & en su lugar?

 Esta advertencia indica que en una expresión de prueba se está probando una variable con dos constantes diferentes y el resultado depende de que se cumpla alguna de las condiciones. Siempre se evalúa como true.

 Normalmente, este problema se debe al uso de `||` en lugar de `&&`, pero también puede deberse al uso de @no__t 2, donde está previsto `==`.

## <a name="example"></a>Ejemplo
 El código siguiente genera esta advertencia:

```
void f(int x)
{
  if ((x != 1) || (x != 3))
  {
    // code
  }
}
```

 Para corregir esta advertencia, utilice el código siguiente:

```
void f(int x)
{
  if ((x != 1) && (x != 3))
  {
    // code
  }
}

/* or */
void f(int x)
{
  if ((x == 1) || (x == 3))
  {
    // code
  }
}
```
