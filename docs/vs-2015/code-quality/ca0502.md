# <a name="ca0502"></a>CA0502

Idioma no admitido.

Este error se produce cuando el usuario intenta analizar un proyecto ASP.NET que contiene uno o más páginas o clases escritas en un lenguaje distinto de C# o Visual Basic. Para corregir este error, quite el archivo de proyecto o deshabilitar el análisis de código.
