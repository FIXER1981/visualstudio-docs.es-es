# <a name="c28226"></a>C28226
C28226 de ADVERTENCIA: La anotación no puede ser un PrimOp (en la declaración actual)

 Esta advertencia indica que la anotación con nombre se declara como PrimOp y también se declaró como una anotación normal. Esto no debería ser posible si las macros estándar se utilizan para las anotaciones. no se espera que esta advertencia se vea en el uso normal.
