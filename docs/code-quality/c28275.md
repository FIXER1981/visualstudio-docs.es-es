# <a name="c28275"></a>C28275
ADVERTENCIA C28275: el parámetro para \_Macro \_value \_ es null

 Esta advertencia indica que hay un error interno en el archivo de modelo, no en el código que se está analizando. Se llamó a la función *macroValue* sin un parámetro.
