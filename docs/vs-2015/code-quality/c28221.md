# <a name="c28221"></a>C28221
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

ADVERTENCIA C28221: se esperaba una expresión de cadena para el parámetro en la anotación  
  
 Esta advertencia indica que se espera que un parámetro de una anotación sea una cadena y se encontró algún otro tipo. Normalmente, esto indica una macro de anotación codificada incorrectamente y no se espera que se vea en el uso normal.
