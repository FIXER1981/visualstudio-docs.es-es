# <a name="c28236"></a>C28236
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

ADVERTENCIA C28236: la anotación no se puede usar en una expresión  
  
 Esta advertencia indica que se ha intentado usar una función declarada como una anotación en un contexto de expresión. Esto no debería ser posible si las macros estándar se utilizan para las anotaciones. no se espera que esta advertencia se vea en el uso típico.
