# <a name="invalid-number"></a>Número no válido
Número no válido (ErrbadNumber 1013)  
  
 Ha intentado usar un número que no es válido en JavaScript.  
  
 Para obtener información sobre los números válidos, vea [Number (objeto](../../javascript/reference/number-object-javascript.md)).
