# <a name="c28024"></a>C28024
C28024 de ADVERTENCIA: El puntero de función al que se asigna se anota con la clase de función \<class >, que no se incluye en las clases de función \<classlist >.

 Esta advertencia se indica cuando ambas funciones se anotaron con una clase de función, pero las clases no coinciden.

 esta advertencia se emite cuando un puntero de función tiene una anotación `_Function_class_` que especifica que solo se deben asignar a ella las funciones de una clase funcional determinada. En una asignación o asignación implícita en una llamada de función, el origen y el destino deben ser de la misma clase de función, pero las clases de función no coinciden.
