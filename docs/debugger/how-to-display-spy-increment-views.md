# <a name="how-to-display-spy-views"></a>Cómo: Mostrar las vistas de Spy++
Cuando se inicia Spy + +, se abre en la [vista de ventanas](../debugger/windows-view.md), una vista de árbol de todas las ventanas y los controles disponibles en la máquina. Esta ventana raíz tiene como título "Windows 1". También hay otras tres vistas de Spy + + disponibles: vista [mensajes](../debugger/messages-view.md), vista [procesos](../debugger/processes-view.md)y [vista subprocesos](../debugger/threads-view.md). Puede mostrar la vista deseada en el menú **Spy** . Para obtener más información, consulte [vistas de Spy + +](../debugger/spy-increment-views.md).
