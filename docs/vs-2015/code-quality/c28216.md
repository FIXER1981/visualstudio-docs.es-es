# <a name="c28216"></a>C28216
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

ADVERTENCIA C28216: la \_ \_ anotación Check_return solo se aplica a las condiciones posteriores del parámetro de función específico.  
  
 La `_Check_return_` anotación se ha aplicado en un contexto distinto de `__post` ; puede necesitar un `__post` modificador (o `__drv_out` ), o bien se puede colocar de forma incorrecta.
