# <a name="c28220"></a>C28220
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Advertencia C28220: Se esperaba una expresión de entero para el parámetro de la anotación  
  
 Esta advertencia indica que se espera un parámetro a una anotación que una expresión de entero y se encontró algún otro tipo. Esto suele indicar una macro de anotación de código incorrecto.
