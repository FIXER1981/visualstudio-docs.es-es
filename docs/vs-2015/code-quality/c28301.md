# <a name="c28301"></a>C28301
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

ADVERTENCIA C28301: no hay anotaciones para la primera declaración de \<function> .\<note1> Vea \<filename> ( \<line> ). \<note2>  
  
 Esta advertencia se indica cuando no se encontraron anotaciones en la primera declaración de una función determinada.
