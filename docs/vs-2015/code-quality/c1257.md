# <a name="c1257"></a>C1257
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Advertencia 1257: Error de inicialización del complemento.  
  
 La herramienta de análisis de código informa de esta advertencia cuando hay un error interno en el complemento, no en el código que se está analizando.
