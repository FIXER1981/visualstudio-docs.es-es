# <a name="c28243"></a>C28243
C28243 de ADVERTENCIA: La anotación para la función del parámetro requiere más desreferenciaciones de las que permite el tipo real anotado

 El número de operadores `__deref` en una anotación es mayor que el número de niveles de puntero definido por el tipo de parámetro. Corrija esta advertencia cambiando el número de la anotación o los niveles de puntero del parámetro al que se hace referencia.
