# <a name="enterprise-mobility--security-ems-in-visual-studio-subscriptions"></a>Beneficio de Enterprise Mobility + Security (EMS) en suscripciones de Visual Studio

Enterprise Mobility + Security, que anteriormente estaba disponible de forma independiente en suscripciones concretas de Visual Studio, ahora se incluye en la suscripción para desarrollador de Microsoft 365 que se incluye en esas mismas suscripciones concretas de Visual Studio.  

Como resultado, Enterprise Mobility + Security ya no se ofrecerá por separado.  

Para activar la suscripción de desarrollador de Microsoft 365:
1. Inicie sesión en el [portal de suscripciones de Visual Studio](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs).
1. Busque el icono de suscripción de desarrollador de Microsoft 365 y haga clic en **Activar**.  
1. Siga los mensajes para completar la creación de la cuenta.  

Obtenga más información sobre la activación de la [suscripción de desarrollador de Microsoft 365](vs-m365.md).
