# <a name="messages-menu-commands"></a>Comandos del menú Mensajes
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

En el menú **Mensajes** se proporcionan comandos para controlar las vistas de mensajes.  
  
 **Iniciar o detener el registro**  
 Inicia o detiene la vista de mensajes.  
  
 **Opciones de registro**  
 Abre el cuadro de diálogo **MessageOptions** , donde puede especificar criterios de búsqueda de mensajes.  
  
 **Borrar registro**  
 Borra el contenido de la vista de mensajes activos.
