## <a name="how-do-i-find-my-admin-if-i-have-a-question"></a>¿Cómo puedo encontrar a mi administrador si tengo una pregunta?

Si le han proporcionado la suscripción en el trabajo o el centro educativo, puede ponerse en contacto con el administrador del contrato para obtener ayuda. Puede usar el formulario ["Contacto con el administrador"](https://my.visualstudio.com/Subscriptions?DisplayContactMyAdminForm=true) directamente desde el [portal de suscripciones](https://my.visualstudio.com/benefits). Para obtener más información, consulte el artículo sobre [Contacto con el administrador de suscripciones](../../../../contact-my-admin.md).
