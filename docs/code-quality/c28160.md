# <a name="c28160"></a>C28160
ADVERTENCIA C28160: anotación de error

 Esta advertencia se indica cuando se ha encontrado una anotación de `__drv_error`. Esta anotación se usa para marcar las prácticas de codificación que se deben corregir y que se pueden usar con una anotación `__drv_when` para indicar combinaciones específicas de parámetros.
