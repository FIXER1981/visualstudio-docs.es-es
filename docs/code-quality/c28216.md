# <a name="c28216"></a>C28216
ADVERTENCIA C28216: el \_comprobación\_devolver\_ anotación solo se aplica a las condiciones posteriores del parámetro de función específico.

 La anotación de `_Check_return_` se ha aplicado en un contexto distinto de `__post`; es posible que necesite un modificador `__post` (o `__drv_out`) o que no se haya colocado correctamente.
