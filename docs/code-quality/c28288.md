# <a name="c28288"></a>C28288
ADVERTENCIA C28288: para la función, error de sintaxis en la anotación de \_At \_ () (nombre de parámetro no válido)

 La herramienta de análisis de código notifica esta advertencia cuando se utiliza la anotación `SAL_at` (`__drv_at`) y la expresión de parámetro no se puede interpretar en el contexto actual. Esto podría incluir el uso de un nombre de miembro o un parámetro mal escrito, o un error ortográfico de palabras clave "Return" o "this".
