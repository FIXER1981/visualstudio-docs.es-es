# <a name="c26461-useconstpointerarguments"></a>C26461 USE_CONST_POINTER_ARGUMENTS:
  El argumento de puntero '% argumento %' para la función '% función %' puede marcarse como puntero a `const`. Consulte [C++ Core Guidelines con.3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-ref).
