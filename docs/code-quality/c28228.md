# <a name="c28228"></a>C28228
ADVERTENCIA C28228: parámetro de anotación: no se puede usar el tipo en las anotaciones

 Esta advertencia indica que un parámetro es de tipo que no se admite. Las anotaciones solo pueden utilizar un conjunto limitado de tipos como parámetros. Esto no debería ser posible si las macros estándar se utilizan para las anotaciones. no se espera que esta advertencia se vea en el uso típico.
