# <a name="c28232"></a>C28232
ADVERTENCIA C28232: \_Pre \_, \_Post \_ o \_Deref no \_ aplicar a ninguna anotación

 Esta advertencia indica que un operador `_Pre_`, `_Post_` o `_Deref_` aparece en una expresión de anotación sin una anotación funcional subsiguiente; se omitió el modificador, pero esto indica una anotación codificada incorrectamente.
