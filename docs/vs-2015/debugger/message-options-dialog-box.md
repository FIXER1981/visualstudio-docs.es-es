# <a name="message-options-dialog-box"></a>Cuadro de diálogo Opciones de mensaje
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Utilice este cuadro de diálogo para seleccionar los mensajes que se enumeran en la [vista Mensajes](../debugger/messages-view.md). Para mostrar este cuadro de diálogo, elija **Mensajes de registro** en el menú de **Spy**.  
  
## <a name="in-this-section"></a>En esta sección  
 [Pestaña Ventanas (Cuadro de diálogo Opciones de mensaje)](../debugger/windows-tab-message-options-dialog-box.md)  
 Se usa para seleccionar los tipos de mensajes que se van a mostrar. Incluye la herramienta de búsqueda.  
  
 [Pestaña Mensajes (Cuadro de diálogo Opciones de mensaje)](../debugger/messages-tab-message-options-dialog-box.md)  
 Se usa para seleccionar los tipos de mensaje que se van a ver.  
  
 [Pestaña Salida (Cuadro de diálogo Opciones de mensaje)](../debugger/output-tab-message-options-dialog-box.md)  
 Se utiliza para especificar qué datos del mensaje se van a mostrar.
