# <a name="idebugprocess2causebreak"></a>IDebugProcess2::CauseBreak
Solicita que el siguiente programa que ejecuta código en este proceso se detenga y envíe un objeto de evento [IDebugBreakEvent2.](../../../extensibility/debugger/reference/idebugbreakevent2.md)

## <a name="syntax"></a>Sintaxis

```cpp
HRESULT CauseBreak( 
   void
);
```

```csharp
int CauseBreak();
```

## <a name="return-value"></a>Valor devuelto
 Si la operación se realiza correctamente, devuelve `S_OK`; de lo contrario, devuelve un código de error.
