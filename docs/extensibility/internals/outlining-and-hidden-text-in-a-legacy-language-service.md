# <a name="outlining-and-hidden-text-in-a-legacy-language-service"></a>Esquematización y texto oculto en un servicio de lenguaje heredado
La esquematización permite contraer una secuencia de líneas de texto en una sola línea. Por ejemplo, en C++ todos los métodos se pueden contraer en una sola línea, mostrando solo la firma del método. El texto oculto es la secuencia de líneas de texto que se pueden mostrar u ocultar.
