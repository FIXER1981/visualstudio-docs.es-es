# <a name="c28222"></a>C28222
ADVERTENCIA 28222: \_sí\_, no \_\_o \_podría\_ esperada para la anotación

 Esta advertencia indica que se espera que un parámetro de una anotación sea uno de los símbolos `_Yes_`, `_No_`o `_Maybe_`y que se encuentre algún otro símbolo. Normalmente, esto indica una macro de anotación codificada incorrectamente.
