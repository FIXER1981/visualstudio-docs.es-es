# <a name="c6385"></a>C6385
ADVERTENCIA C6385: datos no válidos: acceso a \<nombre de búfer >, el tamaño legible es \<tamaño1 > bytes, pero se pueden leer \<tamaño2 > bytes: líneas: x, y

 Esta advertencia indica que la extensión legible del búfer especificado puede ser menor que el índice utilizado para leer de él. Los intentos de leer datos fuera del intervalo válido provocan la saturación del búfer.

## <a name="example"></a>Ejemplo
 El código siguiente genera esta advertencia:

```cpp
void f(int i)
{
   char a[20];
   char j;
   if (i <= 20)
   {
      j = a[i];
   }
}
```

 Para corregir esta advertencia, utilice el código siguiente:

```cpp
void f(int i)
{
   char a[20];
   char j;
   if (i < 20)
   {
      j = a[i];
   }
}
```
