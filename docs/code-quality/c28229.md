# <a name="c28229"></a>C28229
Advertencia C28229: La anotación no admite parámetros

 Esta advertencia indica que se usó una anotación con un parámetro cuando la anotación se declara sin parámetros. Esto no debería ser posible si se utilizan las macros estándares para anotaciones. Esta advertencia no se espera que se muestra un uso típico.
