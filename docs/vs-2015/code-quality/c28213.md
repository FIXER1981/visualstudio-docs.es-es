# <a name="c28213"></a>C28213
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

ADVERTENCIA C28213: se debe utilizar la anotación `_Use_decl_annotations_` para hacer referencia, sin modificación, a una declaración anterior. \<por qué >  
  
 Esta advertencia se indica cuando se hace referencia a una declaración anterior sin la anotación de `_Use_decl_annotations_` necesaria.
