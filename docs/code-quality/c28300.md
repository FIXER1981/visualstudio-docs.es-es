# <a name="c28300"></a>C28300
Advertencia C28300: < nombre de parámetro >: Operandos de expresión de tipos incompatibles para el operador < nombre_operador >

 Esta advertencia se muestra cuando los tipos de operando en un parámetro no son compatibles con el operador.
