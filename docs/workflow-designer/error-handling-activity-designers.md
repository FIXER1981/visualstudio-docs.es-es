# <a name="error-handling-activity-designers"></a>Diseñadores de actividades de control de errores

Los diseñadores de actividades de control de errores se usan para administrar el control de errores en Diseñador de flujo de trabajo. Los temas contenidos en esta sección proporcionan instrucciones sobre cómo usar los diseñadores de actividad de Diseñador de flujo de trabajo y de plantilla.
