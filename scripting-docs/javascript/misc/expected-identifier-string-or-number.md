# <a name="expected-identifier-string-or-number"></a>Se esperaba un identificador, una cadena o un número
Usó una sintaxis literal incorrecta para declarar un literal de objeto. Las propiedades de un literal de objeto deben ser un identificador, una cadena o un número. Un literal de objeto (también denominado "inicializador de objeto") se compone de una lista separada por comas de pares de propiedad: valor, todo ello entre corchetes. Por ejemplo:  
  
```JavaScript  
var point = {x:1.2, y:-3.4};  
```  
  
### <a name="to-correct-this-error"></a>Para corregir este error  
  
- Asegúrese de usar la sintaxis literal adecuada.
