# <a name="c28020"></a>C28020
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

ADVERTENCIA C28020: la expresión \<expr > no es true en esta llamada  
  
 Esta advertencia se indica cuando el \_cumple\_ expresión que se muestra no es true. Con frecuencia, esto indica un parámetro incorrecto.  
  
 Si esto ocurre en una declaración de función, las anotaciones indican una condición imposible.
