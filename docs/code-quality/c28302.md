# <a name="c28302"></a>C28302
C28302 de ADVERTENCIA: En C++ el caso de Reference-Parameter < parameter_name >, se encontró un operador `_Deref_` adicional en \<annotation >.

 Esta advertencia se indica cuando se usa un nivel adicional de `_Deref_` en un parámetro.

 SAL2 no requiere el uso de un nivel adicional de `_Deref_` cuando se trabaja con parámetros de referencia. Esta anotación determinada no es ambigua y se interpreta correctamente, pero debe corregirse.

 Con frecuencia, esto se puede corregir simplemente quitando la anotación `__deref` más antigua y utilizando la sintaxis SAL2. A veces puede ser necesario usar `_At_` para hacer referencia al objeto específico que se va a anotar.
