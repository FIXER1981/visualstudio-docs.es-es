# <a name="c6510"></a>C6510
C6510 de ADVERTENCIA: Anotación no válida: La propiedad ' NullTerminated ' solo se puede usar en búferes cuyos elementos sean de tipo entero o puntero: Función ' \<function > ' \<parameter >.

 Esta advertencia indica un uso incorrecto de la propiedad **NullTerminated** (las que terminan en ' `_z` '). Este tipo de propiedad solo se puede usar en tipos de puntero o matriz.

## <a name="example"></a>Ejemplo
 El código siguiente genera esta advertencia:

```
#include <sal.h>

void f(_In_z_ char x)
{
    // code ...
}
```

 Para corregir esta advertencia, utilice el código siguiente:

```
#include <sal.h>

void f(_In_z_ char * x)
{
    // code ...
}
```
