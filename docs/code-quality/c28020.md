# <a name="c28020"></a>C28020
ADVERTENCIA C28020: la expresión \<expr > no es true en esta llamada

 Esta advertencia se indica cuando no se cumple la expresión de \_ de \_Satisfies que se muestra. Con frecuencia, esto indica un parámetro incorrecto.

 Si esto ocurre en una declaración de función, las anotaciones indican una condición imposible.
