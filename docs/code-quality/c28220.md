# <a name="c28220"></a>C28220
ADVERTENCIA C28220: se esperaba una expresión de entero para la anotación del parámetro en la anotación

 Esta advertencia indica que se espera que un parámetro de una anotación sea una expresión de entero y se encontró algún otro tipo. Normalmente, esto indica una macro de anotación codificada incorrectamente.
