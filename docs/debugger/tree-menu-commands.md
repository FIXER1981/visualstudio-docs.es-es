# <a name="tree-menu-commands"></a>Comandos del menú Árbol
El **árbol** menú proporciona comandos para expandir o contraer las vistas.

 **Expandir un nivel** expande el elemento actualmente seleccionado en una vista del sistema mostrando un nivel por debajo de él.

 **Expandir rama** expande todos los niveles por debajo del elemento seleccionado actualmente.

 **Expandir todo** se expande y muestra todos los niveles en una vista del sistema.

 **Contraer** contrae todos los niveles por debajo del elemento seleccionado actualmente.
