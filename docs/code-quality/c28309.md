# <a name="c28309"></a>C28309
ADVERTENCIA 28309: < parameter_name >: Los operandos de la anotación deben ser tipos Integer/enum/Pointer. No se admiten C++ los operandos void y los operadores sobrecargados. Los valores float se aproximan como enteros. Tipos: @no__t > 0typelist.

 Ha intentado usar un void o una función en una expresión de anotación y el análisis de código no puede administrarlo.  Este error suele producirse cuando se usa un `operator==` que se implementa como una función, pero también se pueden producir otros casos. Examine los tipos de @no__t 0typelist > para obtener pistas sobre lo que es incorrecto.
