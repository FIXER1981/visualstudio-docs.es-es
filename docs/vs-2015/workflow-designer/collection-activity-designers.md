# <a name="collection-activity-designers"></a>Diseñadores de actividades de colecciones
Los diseñadores de actividades de colecciones se utilizan para crear y configurar las actividades que se utilizan para administrar colecciones. Los temas que se incluyen en esta sección y que aparecen en la siguiente tabla ofrecen orientación sobre cómo utilizar la actividad [!INCLUDE[wfd2](../includes/wfd2-md.md)] y diseñadores de plantillas.

## <a name="in-this-section"></a>En esta sección
 [AddToCollection \<T> ](../workflow-designer/addtocollection-t-activity-designer.md) Se utiliza para crear y configurar una <xref:System.Activities.Statements.AddToCollection%601> actividad.

 [ClearCollection \<T> ](../workflow-designer/clearcollection-t-activity-designer.md) Se utiliza para crear y configurar una <xref:System.Activities.Statements.ClearCollection%601> actividad.

 [ExistsInCollection \<T> ](../workflow-designer/existsincollection-t-activity-designer.md) Se utiliza para crear y configurar una <xref:System.Activities.Statements.ExistsInCollection%601> actividad.

 [RemoveFromCollection \<T> ](../workflow-designer/removefromcollection-t-activity-designer.md) Se utiliza para crear y configurar una <xref:System.Activities.Statements.RemoveFromCollection%601> actividad.
