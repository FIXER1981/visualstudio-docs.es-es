# <a name="c28289"></a>C28289
Advertencia C28289: Para que funcione: ReadableTo o WritableTo no tenía una especificación de límite como parámetro

 La herramienta de análisis de código emite esta advertencia cuando se miscoded la anotación del parámetro de función y como se indicó.
