# <a name="c28225"></a>C28225
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

ADVERTENCIA C28225: no se encontró el número correcto de parámetros necesarios en la anotación  
  
 Esta advertencia indica que la anotación con nombre se utiliza con el número de parámetros incorrecto. Esto no debería ser posible si las macros estándar se utilizan para las anotaciones. no se espera que esta advertencia se vea en el uso típico.
