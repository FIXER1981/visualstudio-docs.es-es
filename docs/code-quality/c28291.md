# <a name="c28291"></a>C28291
ADVERTENCIA C28291: post null/NotNull en deref nivel 0 no tiene sentido para la función \<x > en el \<number param >

 La herramienta de análisis de código notifica esta advertencia cuando se especifica la condición post de un parámetro de desreferenciado de nivel cero para tener una propiedad null o no NULL. Este error se produce porque no se puede cambiar un valor en el nivel de desreferencia cero.
