# <a name="ijsdebugframegetname-method"></a>IJsDebugFrame::GetName (Método)
Obtiene el nombre descriptivo del marco de pila.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp
HRESULT GetName(  
   BSTR *pName  
);  
```  
  
#### <a name="parameters"></a>Parámetros  
 `pName`  
 [out] Nombre del marco de pila.  
  
## <a name="return-value"></a>Valor devuelto  
  
## <a name="requirements"></a>Requisitos  
 **Encabezado:** jscript9diag. h
