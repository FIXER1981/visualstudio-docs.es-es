# <a name="c28351"></a>C28351
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

ADVERTENCIA C28351: la anotación \<annotation> describe dónde no se puede usar un valor dinámico (una variable) en la condición.  
  
 Esta advertencia se indica cuando se aplica una anotación en la que se inspecciona el tipo de C/C++.  
  
 La expresión de `_When_` debe evaluarse como una constante. `_When_`Se omite.
