# <a name="ca2322-ensure-javascriptserializer-is-not-initialized-with-simpletyperesolver-before-deserializing"></a>CA2322: Asegúrese de que JavaScriptSerializer no se ha inicializado con SimpleTypeResolver antes de la deserialización.

|||
|-|-|
|TypeName|EnsureJavaScriptSerializerIsNotInitializedWithSimpleTypeResolverBeforeDeserializing|
|Identificador de comprobación|CA2322|
|Categoría|Microsoft.Security|
|Cambio importante|Poco problemático|

## <a name="cause"></a>Motivo

Se llamó a <xref:System.Web.Script.Serialization.JavaScriptSerializer?displayProperty=nameWithType> método de deserialización o se hizo referencia a él y el <xref:System.Web.Script.Serialization.JavaScriptSerializer> puede haberse inicializado con una <xref:System.Web.Script.Serialization.SimpleTypeResolver?displayProperty=nameWithType>.

## <a name="rule-description"></a>Descripción de la regla

[!INCLUDE[insecure-deserializers-description](includes/insecure-deserializers-description-md.md)]

Esta regla busca <xref:System.Web.Script.Serialization.JavaScriptSerializer?displayProperty=nameWithType> llamadas o referencias al método de deserialización, cuando es posible que el <xref:System.Web.Script.Serialization.JavaScriptSerializer> se haya inicializado con una <xref:System.Web.Script.Serialization.SimpleTypeResolver?displayProperty=nameWithType>.

## <a name="how-to-fix-violations"></a>Cómo corregir infracciones

- Asegúrese de que <xref:System.Web.Script.Serialization.JavaScriptTypeResolver?displayPropertyName=nameWithType> objetos no se inicializan con un <xref:System.Web.Script.Serialization.SimpleTypeResolver?displayProperty=nameWithType>.
- Si el código necesita leer los datos serializados mediante un <xref:System.Web.Script.Serialization.SimpleTypeResolver>, restrinja los tipos deserializados a una lista de espera implementando un <xref:System.Web.Script.Serialization.JavaScriptTypeResolver> personalizado.
- Convierta la prueba de alteración de los datos serializados. Después de la serialización, firme criptográficamente los datos serializados. Antes de la deserialización, valide la firma criptográfica. Proteja la clave criptográfica para que no se revele y diseñe las rotaciones de clave.

## <a name="pseudo-code-examples"></a>Ejemplos de pseudocódigo

### <a name="violation"></a>Infracción

```csharp
using System.Web.Script.Serialization;

public class ExampleClass
{
    public JavaScriptSerializer Serializer { get; set; }

    public T Deserialize<T>(string str)
    {
        return this.Serializer.Deserialize<T>(str);
    }
}
```

```vb
Imports System.Web.Script.Serialization

Public Class ExampleClass
    Public Property Serializer As JavaScriptSerializer

    Public Function Deserialize(Of T)(str As String) As T
        Return Me.Serializer.Deserialize(Of T)(str)
    End Function
End Class
```

### <a name="solution"></a>Soluciones

```csharp
using System.Web.Script.Serialization;

public class ExampleClass
{
    public T Deserialize<T>(string str)
    {
        JavaScriptSerializer s = new JavaScriptSerializer();
        return s.Deserialize<T>(str);
    }
}
```

```vb
Imports System.Web.Script.Serialization

Public Class ExampleClass
    Public Function Deserialize(Of T)(str As String) As T
        Dim s As JavaScriptSerializer = New JavaScriptSerializer()
        Return s.Deserialize(Of T)(str)
    End Function
End Class
```

### <a name="solution"></a>Soluciones

```csharp
using System;
using System.Web.Script.Serialization;

public class BookRecordTypeResolver : JavaScriptTypeResolver
{
    // For compatibility with data serialized with a JavaScriptSerializer initialized with SimpleTypeResolver.
    private static readonly SimpleTypeResolver Simple = new SimpleTypeResolver();

    public override Type ResolveType(string id)
    {
        // One way to discover expected types is through testing deserialization
        // of **valid** data and logging the types used.

        ////Console.WriteLine($"ResolveType('{id}')");

        if (id == typeof(BookRecord).AssemblyQualifiedName || id == typeof(AisleLocation).AssemblyQualifiedName)
        {
            return Simple.ResolveType(id);
        }
        else
        {
            throw new ArgumentException("Unexpected type ID", nameof(id));
        }
    }

    public override string ResolveTypeId(Type type)
    {
        return Simple.ResolveTypeId(type);
    }
}

public class BookRecord
{
    public string Title { get; set; }
    public string Author { get; set; }
    public int PageCount { get; set; }
    public AisleLocation Location { get; set; }
}

public class AisleLocation
{
    public char Aisle { get; set; }
    public byte Shelf { get; set; }
}

public class ExampleClass
{
    public BookRecord DeserializeBookRecord(string s)
    {
        JavaScriptSerializer serializer = new JavaScriptSerializer(new BookRecordTypeResolver());
        return serializer.Deserialize<BookRecord>(s);
    }
}
```

```vb

    Private Dim Simple As SimpleTypeResolver = New SimpleTypeResolver()

        ' of **valid** data and logging the types used.

    End Function
```
