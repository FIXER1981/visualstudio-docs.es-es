# <a name="c28279"></a>C28279
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

ADVERTENCIA C28279: para el símbolo, se encontró un ' begin ' sin un ' End ' correspondiente  
  
 El lenguaje de anotación admite una construcción Begin y end ( `{` y `}` en C), y el emparejamiento se ha desequilibrado. Esta situación se puede evitar si se usan las macros.
