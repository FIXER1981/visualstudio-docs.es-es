# <a name="enabling-the-azure-sdk"></a>Habilitación de Azure SDK
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

El SDK de Azure para .NET le ayuda a crear, configurar, compilar, depurar, ejecutar y empaquetar aplicaciones y servicios web escalables en Azure.  
  
 Puede descargar el SDK de Azure desde el cuadro de diálogo **nuevo proyecto** .  
  
 [!INCLUDE[note_settings_general](../includes/note-settings-general-md.md)]  
  
### <a name="to-enable-the-azure-sdk"></a>Para habilitar el SDK de Azure  
  
1. En el menú **archivo** , cree un nuevo proyecto. (Haga clic en **nuevo** y, a continuación, en **proyecto**).  
  
2. Expanda el nodo de **Visual Basic** o **C#** y, a continuación, haga clic en **nube**.  
  
3. En el panel central, seleccione **obtener SDK de Azure para .net**y, a continuación, haga clic en **Aceptar**.  
  
     Siga las instrucciones que aparecen en el instalador.  
  
    > [!NOTE]
    > Para instalar las herramientas de Azure, debe tener instalados Internet Information Services 7,0 y los componentes de desarrollo de aplicaciones de ASP.NET. Si estos componentes no están instalados, el programa de instalación de Azure Tools le pedirá que los instale.
