# <a name="c28244"></a>C28244
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Advertencia C28244: La anotación de función tiene una anotación de parámetros o externa no se puede analizar  
  
 Esto actualmente no debería ser posible si se utilizan las macros estándares para anotaciones. Esta advertencia no se espera que se muestra un uso típico.
