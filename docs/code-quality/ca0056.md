# <a name="ca0056"></a>CA0056
Un archivo de proyecto tiene una versión incorrecta de la herramienta de análisis.

Esta advertencia se produce si se invoca FxCopCmd. exe manualmente con un archivo de proyecto que tiene una versión incorrecta de la herramienta de análisis de código.
