# <a name="c28212"></a>C28212
ADVERTENCIA C28212: se esperaba una expresión de puntero para la anotación

 Esta advertencia indica que se espera que el parámetro numerado de una anotación (no la función que se va a anotar) sea un puntero o un tipo de matriz, pero se encontró algún otro tipo. La anotación debe corregirse.
