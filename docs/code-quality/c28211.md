# <a name="c28211"></a>C28211
Advertencia C28211: Se esperaba un nombre de contexto estático para SAL_context

 Esta advertencia indica que el operando situado a la `_Static_context_` anotación debe ser el nombre de un contexto definido por la herramienta. Esta advertencia nunca debe producirse en el uso normal.
