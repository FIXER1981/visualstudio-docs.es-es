# <a name="color-theming-tools"></a>Herramientas de creación de temas de color
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

El SDK de VS proporciona varias herramientas que le ayudarán a crear y administrar temas de color.  
  
- La herramienta [Editor de colores de VSIX](../../extensibility/internals/vsix-color-editor.md) puede crear y editar colores personalizados para Visual Studio.  
  
- La herramienta de [compilador de color VSIX](../../extensibility/internals/vsix-color-compiler.md) convierte los archivos. XML de tema de Visual Studio existentes en. pkgdef para la extensión.
