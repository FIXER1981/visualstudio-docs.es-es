# <a name="c28229"></a>C28229
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

ADVERTENCIA C28229: la anotación no admite parámetros  
  
 Esta advertencia indica que se ha utilizado una anotación con un parámetro cuando la anotación se declara sin parámetros. Esto no debería ser posible si las macros estándar se utilizan para las anotaciones. no se espera que esta advertencia se vea en el uso típico.
