# <a name="c28196"></a>C28196
C28196 de ADVERTENCIA: No se cumple el requisito. (La expresión no se evalúa como true).

 Esta advertencia indica que el prototipo de función de la función que se está analizando tiene un `__notnull`, `__null` o `__drv_valueIs` en un parámetro `_Out_` o el valor devuelto, pero el valor devuelto no es coherente con esa anotación.
