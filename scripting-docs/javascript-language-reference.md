# <a name="javascript-language-reference"></a>Referencia de lenguaje de JavaScript

JavaScript es un lenguaje de scripting que se puede insertar en páginas web y otras aplicaciones.  
  
La implementación de JavaScript de Microsoft es compatible con la versión 5 de la especificación del lenguaje ECMAScript. También proporciona características adicionales no incluidas en las normas ECMA.

> [!IMPORTANT]
> Microsoft se ha unido al esfuerzo de toda la comunidad de enviar [documentos web de MDN](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference) a la referencia de desarrollo de un punto de partida de la web, que comienza redirigiendo las 500 páginas de la referencia de la API de JavaScript de Microsoft de [docs.Microsoft.com](https://docs.microsoft.com) a sus homólogos de MDN y portando nuestra referencia de extensiones de JS heredada para fines históricos y de servicio.
