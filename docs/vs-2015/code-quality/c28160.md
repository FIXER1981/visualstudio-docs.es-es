# <a name="c28160"></a>C28160
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

ADVERTENCIA C28160: anotación de error  
  
 Esta advertencia se indica cuando se ha `__drv_error` encontrado una anotación. Esta anotación se usa para marcar las prácticas de codificación que se deben corregir y se puede usar con una `__drv_when` anotación para indicar combinaciones específicas de parámetros.
