# <a name="idebugexpressionevaluatorsetlocale"></a>IDebugExpressionEvaluator::SetLocale
Este método establece el idioma que se usará para crear resultados imprimibles.

## <a name="syntax"></a>Sintaxis

```cpp
HRESULT SetLocale( 
   WORD wLangID
);
```

```csharp
int SetLocale(
   ushort wLangID
);
```

## <a name="parameters"></a>Parámetros
`wLangID`\
[en] El identificador de idioma.

## <a name="return-value"></a>Valor devuelto
 Si la operación se realiza correctamente, devuelve `S_OK`; de lo contrario, devuelve un código de error.

## <a name="remarks"></a>Observaciones
 Este método se puede llamar muchas veces mientras se carga el evaluador de expresiones (EE), por lo que el EE debe ser capaz de cambiar los idiomas sobre la marcha. EE utiliza esta configuración regional para devolver mensajes de error y cadenas en el idioma adecuado.
