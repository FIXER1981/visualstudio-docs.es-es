# <a name="ienumremotedebugapplications-interface"></a>IEnumRemoteDebugApplications (Interfaz)
Enumera los objetos de la aplicación. Esta interfaz se puede usar para enumerar las aplicaciones en ejecución en un equipo para un cuadro de diálogo "asociar a la aplicación".  
  
 Además de los métodos heredados de `IUnknown`, el `IEnumRemoteDebugApplications` interfaz expone los métodos siguientes.  
  
## <a name="methods-in-vtable-order"></a>Métodos en orden de Vtable  
  
|Método|Descripción|  
|------------|-----------------|  
|[IEnumRemoteDebugApplications::Clone](../../winscript/reference/ienumremotedebugapplications-clone.md)|Crea un enumerador que contiene el mismo estado que el enumerador actual.|  
|[IEnumRemoteDebugApplications::Next](../../winscript/reference/ienumremotedebugapplications-next.md)|Recupera un número especificado de segmentos en una secuencia de enumeración.|  
|[IEnumRemoteDebugApplications::Reset](../../winscript/reference/ienumremotedebugapplications-reset.md)|Restablece una secuencia de enumeración al principio.|  
|[IEnumRemoteDebugApplications::Skip](../../winscript/reference/ienumremotedebugapplications-skip.md)|Omite un número especificado de segmentos en una secuencia de enumeración.|
