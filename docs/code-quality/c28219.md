# <a name="c28219"></a>C28219
C28219 de ADVERTENCIA: Se esperaba un miembro de enumeración para el parámetro de la anotación

 Se espera que un parámetro de una anotación sea miembro del tipo `enum` con nombre y se haya encontrado algún otro símbolo; Use un miembro de ese tipo `enum`. Normalmente, esto indica una macro de anotación codificada incorrectamente.
