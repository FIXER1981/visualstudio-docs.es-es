# <a name="c28208"></a>C28208
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

warning C28208: Función \<función > se ha definido con una lista de parámetros distinta en \<archivo > (\<línea >). Algunas herramientas de análisis tendrán resultados incorrectos  
  
 Esta advertencia se muestra cuando la definición conocidos de la función no coincide con otra aparición.
