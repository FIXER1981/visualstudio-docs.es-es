# <a name="c28273"></a>C28273
Advertencia C28273: Para la función, las pistas son incoherentes con la declaración de la función

 Esta advertencia indica un error en las anotaciones, no en el código que se está analizando. Las anotaciones que aparecen en una definición de función no son coherentes con los que aparecen en una declaración. Las dos anotaciones deben resolverse para que coincida con.
