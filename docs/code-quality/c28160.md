# <a name="c28160"></a>C28160
C28160 de ADVERTENCIA: Error (anotación)

 Esta advertencia se indica cuando se ha encontrado una anotación `__drv_error`. Esta anotación se usa para marcar las prácticas de codificación que se deben corregir y se puede usar con una anotación `__drv_when` para indicar combinaciones específicas de parámetros.
