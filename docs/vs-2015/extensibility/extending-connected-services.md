# <a name="extending-connected-services"></a>Extensión de Servicios conectados
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

El SDK de Servicios conectados de Visual Studio permite a los proveedores de servicios crear extensiones de Visual Studio que los desarrolladores de aplicaciones pueden agregar fácilmente a su proyecto sin tener que cerrar el IDE de Visual Studio.
