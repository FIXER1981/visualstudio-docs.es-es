# <a name="c28204"></a>C28204
Advertencia C28204: \<función >: Solo una de estas sobrecargas y el de uno en \<filename > (\<línea >) se anotan para \<paramname >: se debe anotar ambas o ninguna de ellas.

 Esta advertencia se muestra cuando se produce un error en las anotaciones.
