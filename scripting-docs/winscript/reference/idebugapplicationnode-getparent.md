# <a name="idebugapplicationnodegetparent"></a>IDebugApplicationNode::GetParent
Devuelve el nodo primario de este nodo de aplicación.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp
HRESULT GetParent(  
   IDebugApplicationNode**  pprddp  
);  
```  
  
#### <a name="parameters"></a>Parámetros  
 `pprddp`  
 enuncia Nodo de aplicación principal de este nodo de aplicación.  
  
## <a name="return-value"></a>Valor devuelto  
 El método devuelve un objeto `HRESULT`. Entre los valores posibles se incluyen los que se indican en la tabla siguiente, entre otros.  
  
|Valor|Descripción|  
|-----------|-----------------|  
|`S_OK`|El método se realizó correctamente.|  
  
## <a name="remarks"></a>Comentarios  
 Este método devuelve el nodo primario de este nodo de aplicación.
