# <a name="c26466-no_static_downcast_polymorphic"></a>C26466 NO_STATIC_DOWNCAST_POLYMORPHIC
  No utilice `static_cast` downcasts. Una conversión de un tipo polimórfico debe usar dynamic_cast. Consulte [C++ Core Guidelines Type.2](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Pro-type-downcast).
