# <a name="help-menu-commands"></a>Comandos del menú Ayuda
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

El menú **Ayuda** proporciona comandos para abrir archivos de ayuda.  
  
 **Index**  
 Abre la pantalla de contenido de la ayuda de Spy + +.  
  
 **Uso de la Ayuda**  
 Abre un archivo de ayuda que describe cómo usar la ayuda de.  
  
 **Acerca de Spy++**  
 Muestra la versión de software y la información de copyright de Spy + +.
