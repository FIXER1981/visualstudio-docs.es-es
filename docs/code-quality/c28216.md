# <a name="c28216"></a>C28216
C28216 de ADVERTENCIA: La anotación \_Check @ no__t-1return @ no__t-2 solo se aplica a las condiciones posteriores del parámetro de función específico.

 La anotación `_Check_return_` se ha aplicado en un contexto distinto de `__post`; es posible que necesite un modificador `__post` (o `__drv_out`) o que no se haya colocado correctamente.
