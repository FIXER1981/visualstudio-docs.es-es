# <a name="c28243"></a>C28243
ADVERTENCIA C28243: la anotación para la función en el parámetro requiere más desreferencias de las que permite el tipo real anotado

 El número de operadores de `__deref` en una anotación es mayor que el número de niveles de puntero definido por el tipo de parámetro. Corrija esta advertencia cambiando el número de la anotación o los niveles de puntero del parámetro al que se hace referencia.
