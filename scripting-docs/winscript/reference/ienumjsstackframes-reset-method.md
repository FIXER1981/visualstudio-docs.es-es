# <a name="ienumjsstackframesreset-method"></a>IEnumJsStackFrames::Reset (Método)
Restablece el marco de pila a la posición anterior al primer elemento.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp
HRESULT Reset(void);  
```
