# <a name="script_e_propagate-error-code"></a>SCRIPT_E_PROPAGATE (Código de error)
Se propagará un error de script al autor de la llamada, que podría estar en un subproceso diferente. El host debe pasar el código de error al autor de la llamada.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
