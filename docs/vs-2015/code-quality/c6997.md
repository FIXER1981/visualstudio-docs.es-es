# <a name="c6997"></a>C6997
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Advertencia 6997: Las anotaciones de esta ubicación carecen de sentido y se omitirá.  
  
 Las anotaciones no se pueden aplicar a `extern "C" {...}`. Aplique las anotaciones a un objeto específico.
