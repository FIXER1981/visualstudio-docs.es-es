# <a name="c28246"></a>C28246
Advertencia C28246: La anotación de la función '\<nombre >'-parámetro '\<parámetro >' no coincide con el tipo del parámetro

 Un operador __deref se aplicó a un tipo que no son de puntero al crear una anotación.
