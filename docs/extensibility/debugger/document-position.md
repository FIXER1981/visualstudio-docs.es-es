# <a name="document-position"></a>Posición del documento
En [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] la depuración, una posición de *documento:*

- Proporciona una abstracción de una posición en un archivo de origen como se conoce al IDE. Para la mayoría de los idiomas actuales, una posición de documento se puede considerar como una posición en un archivo de origen.

- Describe una posición en un documento de origen en un motor de depuración.

- Se implementa mediante un [IDebugDocumentPosition2](../../extensibility/debugger/reference/idebugdocumentposition2.md) interfaz.
