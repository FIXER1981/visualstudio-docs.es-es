# <a name="c28211"></a>C28211
ADVERTENCIA C28211: se esperaba un nombre de contexto estático para SAL_context

 Esta advertencia indica que el operando de la anotación `_Static_context_` debe ser el nombre de un contexto definido por la herramienta. Esta advertencia nunca debe producirse en el uso normal.
