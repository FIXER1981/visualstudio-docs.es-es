# <a name="c28286"></a>C28286
C28286 de ADVERTENCIA: Para la función, error de sintaxis cerca del final

 La herramienta de análisis de código notifica esta advertencia cuando se encuentra un error probable en el archivo de modelo. Algunos errores de archivos de código fuente también pueden provocar estos errores.
