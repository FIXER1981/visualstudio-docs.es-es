# <a name="c28208"></a>C28208
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

ADVERTENCIA C28208: \<function> la función se definió previamente con una lista de parámetros diferente en \<file> ( \<line> ). Algunas herramientas de análisis producirán resultados incorrectos  
  
 Esta advertencia se indica cuando la definición conocida de una función no coincide con otra repetición.
