# <a name="c28021"></a>C28021
ADVERTENCIA C28021: el parámetro \<parámetro > que se va a anotar con \<Anno > debe ser un puntero

 Esta advertencia se indica cuando el objeto que se va a anotar no es un tipo de puntero. Esta anotación no se puede usar con `void` o tipos enteros.
