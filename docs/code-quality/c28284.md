# <a name="c28284"></a>C28284
C28284 de ADVERTENCIA: En el caso de Symbol, actualmente no se admiten predicados en símbolos que no son de función

 Esta advertencia indica que se encontró una anotación condicional (predicado, `__drv_when`) en algo que no es una función.
