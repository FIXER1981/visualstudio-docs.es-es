# <a name="ijsdebug-interface"></a>IJsDebug (Interfaz)
Punto de entrada para la depuración de JavaScript.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp
IJsDebug : public IUnknown;  
```
