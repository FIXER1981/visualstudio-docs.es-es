# <a name="c28246"></a>C28246
C28246 de ADVERTENCIA: La anotación de la función ' \<name > '-Parameter ' \<parameter > ' no coincide con el tipo del parámetro

 Se aplicó un operador __deref a un tipo que no es de puntero al crear una anotación.
