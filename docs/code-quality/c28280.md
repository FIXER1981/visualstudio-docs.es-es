# <a name="c28280"></a>C28280
C28280 de ADVERTENCIA: Para el símbolo, se encontró un 'end' sin un 'begin' coincidente

 El lenguaje de anotación admite una construcción Begin y end (`{` y `}` in C) y el emparejamiento se ha desequilibrado. Esta situación se puede evitar si se usan las macros.
