# <a name="c26460-useconstreferencearguments"></a>C26460 USE_CONST_REFERENCE_ARGUMENTS

  El argumento de referencia '% argumento %' para la función "% función %" se puede marcar como `const`. Consulte [C++ Core Guidelines con.3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-ref).
