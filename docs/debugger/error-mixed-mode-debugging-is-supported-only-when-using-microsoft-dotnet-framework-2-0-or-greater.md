# <a name="error-mixed-mode-debugging-is-supported-only-when-using-microsoft-net-framework-20-or-greater"></a>Error: El modo de depuración mixto se admite solo cuando se usa Microsoft .NET Framework 2.0 o superior
Para depurar código nativo y administrado mixto, debe tener .NET Framework versión 2.0, 3.0. 3.5 ó 4. No se admite la depuración con versiones anteriores de .NET Framework en modo mixto.

### <a name="to-correct-this-error"></a>Para corregir este error

- Actualizar .NET Framework versión 2.0, 3.0, 3.5 o 4.0.
