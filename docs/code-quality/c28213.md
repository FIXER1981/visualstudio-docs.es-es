# <a name="c28213"></a>C28213
C28213 de ADVERTENCIA: La anotación `_Use_decl_annotations_` debe usarse para hacer referencia, sin modificación, a una declaración anterior. @no__t 0why >

 Esta advertencia se indica cuando se hace referencia a una declaración anterior sin la anotación `_Use_decl_annotations_` requerida.
