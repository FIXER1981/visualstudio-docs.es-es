# <a name="c6274"></a>C6274
ADVERTENCIA C6274: no se ha pasado ningún carácter como parámetro \<number > cuando se requiere un carácter en la llamada a \<Function >

 Esta advertencia indica que la cadena de formato especifica que se requiere un carácter (por ejemplo, una especificación `%c` o `%C`), pero se pasa un valor no entero como Float, String o struct. Es probable que este defecto cause resultados incorrectos.

## <a name="example"></a>Ejemplo
 El código siguiente genera esta advertencia:

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str);
}
```

 Para corregir esta advertencia, utilice el código siguiente:

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str[0]);
}
```

 En el código siguiente se usa la función de manipulación de cadenas seguras, `sprintf_s`, para corregir esta ADVERTENCIA:

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf_s(buff,5,"%c", str[0]);
}
```
