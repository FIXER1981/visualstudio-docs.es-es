# <a name="migration-activity-designers"></a>Diseñadores de actividades de migraciones

Los diseñadores de actividad de migración se utilizan para llamar al flujo de trabajo 3.0/3.5 actividades desde un flujo de trabajo como destino .NET Framework 4. Los temas contenidos en esta sección proporcionan instrucciones sobre cómo usar los diseñadores de actividad del Diseñador de flujo de trabajo.
