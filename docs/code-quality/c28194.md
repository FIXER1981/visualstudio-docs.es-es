# <a name="c28194"></a>C28194
C28194 de ADVERTENCIA: La función se declaró como un alias del valor de la variable y salió sin hacerlo.

 Esta advertencia indica que el prototipo de función de la función que se está analizando tiene una anotación `__drv_isAliased`, que indica que se creará un *alias* para el argumento especificado (es decir, se asignará el valor de tal forma que sobreviva el retorno de la función). Sin embargo, la función no agrega alias al argumento a lo largo de la ruta de acceso indicada por la anotación. La mayoría de las funciones que alias a una variable guardan su valor en una estructura de datos global.
