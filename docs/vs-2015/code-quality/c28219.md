# <a name="c28219"></a>C28219
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

ADVERTENCIA C28219: miembro de la enumeración esperado para la anotación del parámetro en la anotación  
  
 Se espera que un parámetro de una anotación sea un miembro del tipo con nombre `enum` y se encontró algún otro símbolo; use un miembro de ese `enum` tipo. Normalmente, esto indica una macro de anotación codificada incorrectamente.
