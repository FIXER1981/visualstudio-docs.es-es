# <a name="expected-var"></a>Se esperaba 'var'
Se esperaba ' var ' (ERRnoVar 1034))  
  
 No se pudo agregar la palabra clave var cuando era necesaria.  
  
 Para obtener más información sobre la palabra clave var, vea [instrucción var](../../javascript/reference/var-statement-javascript.md).
