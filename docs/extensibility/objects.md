# <a name="objects"></a>de la empresa
La siguiente información de referencia explica los objetos que se usan para crear una instancia del editor de texto principal.

## <a name="in-this-section"></a>En esta sección
- [VSCodeWindow (objeto)](../extensibility/vscodewindow-object.md) describe el objeto de ventana de código, una ventana de documento especializado que puede incluir una o varias vistas de texto.

- [VSCodeWindowManager (objeto)](../extensibility/vscodewindowmanager-object.md) describe el objeto de administrador de ventana de código que se implementa mediante el servicio de lenguaje y es responsable de administrar los elementos gráficos.

- [VSTextBuffer (objeto)](../extensibility/vstextbuffer-object.md) describe el objeto de búfer de texto, que representa una secuencia de texto Unicode, los datos de un archivo.

- [VSTextView (objeto)](../extensibility/vstextview-object.md) describe la vista de texto, una ventana que permite a los usuarios ver y editar texto Unicode del búfer de texto.
