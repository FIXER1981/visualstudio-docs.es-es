# <a name="c28222"></a>C28222
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

ADVERTENCIA 28222: \_ sí \_ , \_ no \_ o \_ quizás \_ se esperaba para la anotación  
  
 Esta advertencia indica que se espera que un parámetro de una anotación sea uno de los símbolos `_Yes_` , `_No_` o `_Maybe_` , y se encontró algún otro símbolo. Normalmente, esto indica una macro de anotación codificada incorrectamente.
