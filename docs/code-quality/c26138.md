# <a name="c26138"></a>C26138
ADVERTENCIA C26138: suspendiendo una corutina mientras mantiene > de bloqueo \<lock.

Warning C26138 advierte cuando se suspende una corutina mientras se mantiene un bloqueo. En general, no se puede saber cuánto tiempo permanecerá una corutina en el estado suspendido, por lo que este patrón puede dar lugar a secciones críticas más largas de las esperadas.

## <a name="example"></a>Ejemplo
 El código siguiente generará C26138.

```cpp

#include <experimental/generator>
#include <future>
#include <mutex>

using namespace std::experimental;

std::mutex global_m;
_Guarded_by_(global_m) int var = 0;

generator<int> mutex_acquiring_generator() {
  global_m.lock();
  ++var;
  co_yield 1;                           // @expected(26138), global_m is hold while yielding.
  global_m.unlock();
}

generator<int> mutex_acquiring_generator_report_once() {
  global_m.lock();
  ++var;
  co_yield 1;                           // @expected(26138), global_m is hold while yielding.
  co_yield 1;                           // @expected(26138), global_m is hold while yielding.
  global_m.unlock();
}
```

## <a name="example"></a>Ejemplo
 El código siguiente corregirá estas advertencias.

```cpp

#include <experimental/generator>
#include <future>
#include <mutex>

using namespace std::experimental;

std::mutex global_m;
_Guarded_by_(global_m) int var = 0;

generator<int> mutex_acquiring_generator2() {
  {
    global_m.lock();
    ++var;
    global_m.unlock();
  }
  co_yield 1;                           // no 26138, global_m is already released above.
}
```
