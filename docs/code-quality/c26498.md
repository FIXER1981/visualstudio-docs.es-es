# <a name="c26498-use_constexpr_for_functioncall"></a>C26498 USE_CONSTEXPR_FOR_FUNCTIONCALL

Esta llamada de función% FUNCTION% puede usar `constexpr` si se desea la evaluación en tiempo de compilación. Consulte [C++ Core Guidelines con.5](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-constexpr).
