# <a name="c1254"></a>C1254
ADVERTENCIA C1254: no coincide la versión del complemento: la versión no coincide con la versión del controlador de PREfast

 La herramienta de análisis de código notifica esta advertencia cuando se produce un error interno con la versión del complemento, no en el código que se está analizando.
