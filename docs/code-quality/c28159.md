# <a name="c28159"></a>C28159
C28159 de ADVERTENCIA: Considere la posibilidad de usar otra función en su lugar.

 Esta advertencia se indica para que los controladores sugieran que se use una llamada de función preferida que sea semánticamente equivalente a la función a la que llama el controlador. Se trata de un mensaje de advertencia general; se usó la anotación `__drv_preferredFunction` (posiblemente con una anotación condicional `__drv_when` ()) para marcar una práctica de codificación incorrecta.

## <a name="example"></a>Ejemplo
 El siguiente código de ejemplo genera esta advertencia:

```
char buff[MAX_PATH];

OemToChar(buff, input);

    // if strlen(input) > MAX_PATH
....// leads to buffer overrun
```

 El siguiente código de ejemplo evita esta advertencia:

```
char buff[MAX_PATH];

OemToCharBuff(buff, input, MAX_PATH);
```
