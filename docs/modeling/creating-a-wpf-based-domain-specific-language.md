Puede encontrar información y ejemplos acerca de esta característica en el sitio web de herramientas de visualización y modelado de Visual Studio en [http://go.microsoft.com/fwlink/?LinkId=186128](https://code.msdn.microsoft.com/Visualization-and-Modeling-313535db)
