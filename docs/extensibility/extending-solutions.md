# <a name="extend-solutions"></a>Ampliar las soluciones
Una solución es una agrupación de uno o varios proyectos que trabajan juntos para crear una aplicación.

## <a name="in-this-section"></a>En esta sección
 En los siguientes tutoriales se muestran algunas formas en las que puede ampliar las soluciones.

1. [Extender el filtro Explorador](../extensibility/extending-the-solution-explorer-filter.md) de soluciones explica cómo filtrar lo que muestra el Explorador de soluciones.

2. Administrar la carga de [proyectos en una solución](../extensibility/managing-project-loading-in-a-solution.md) explica cómo administrar la carga de proyectos para mejorar el rendimiento cuando un usuario carga una solución.
