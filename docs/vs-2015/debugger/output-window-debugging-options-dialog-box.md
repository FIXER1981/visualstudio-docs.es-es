# <a name="output-window-debugging-options-dialog-box"></a>Ventana de salida, Depuración, Opciones (Cuadro de diálogo)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Puede especificar qué tipos de información de depuración aparecen en la **Ventana de salida**. Para mostrar estas opciones, abra el menú **Herramientas**, haga clic en **Opciones**, expanda el nodo **Depuración** y haga clic en **Ventana de salida**.  
  
 **Configuración general de salida**  
 Esta categoría contiene controles que determinan si los mensajes de depuración generales aparecen en la **salida** ventana. Puede especificar si cada tipo de mensaje aparecerá o no.  
  
 **Configuración de seguimiento de WPF**  
 Esta categoría contiene controles que determinan el nivel de mensajes de seguimiento WPF que aparecen en la **salida** ventana. Puede especificar si cada tipo de mensaje aparecerá y especificar un nivel que va de **Crítico** a **Todos**.  
  
 Para obtener más información, consulte [Cómo Presentación de información de seguimiento de WPF](../debugger/how-to-display-wpf-trace-information.md).
