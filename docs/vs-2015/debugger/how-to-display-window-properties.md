# <a name="how-to-display-window-properties"></a>Filtrar Presentación de propiedades de ventana
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Para obtener más información acerca de una ventana que aparece en [Windows Vista](../debugger/windows-view.md), utilice el [cuadro de diálogo de propiedades de ventana](../debugger/window-properties-dialog-box.md).  
  
### <a name="to-open-the-window-properties-dialog-box"></a>Para abrir el cuadro de diálogo Propiedades de la ventana  
  
1. Seleccione el elemento y luego elija **propiedades** desde el **vista** menú.  
  
   Cuadros de diálogo de propiedades no son modales. Al hacer clic en los elementos en una ventana de vista, se actualiza automáticamente el cuadro de diálogo con información sobre cada elemento seleccionado.
