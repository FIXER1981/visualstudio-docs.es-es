# <a name="iactivescriptproperty"></a>IActiveScriptProperty
El `IActiveScriptProperty` interfaz se usa para obtener y establecer las propiedades de configuración.  
  
## <a name="methods-in-vtable-order"></a>Métodos en orden de Vtable  
  
|Método|Descripción|  
|------------|-----------------|  
|[GetProperty](../../winscript/reference/iactivescriptproperty-getproperty.md)|Obtiene la propiedad especificada por el parámetro.|  
|[SetProperty](../../winscript/reference/iactivescriptproperty-setproperty.md)|Establece la propiedad especificada por el parámetro.|
