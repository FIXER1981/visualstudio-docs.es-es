# <a name="c28230"></a>C28230
C28230 de ADVERTENCIA: El tipo de parámetro no tiene ningún miembro.

 Esta advertencia indica que un argumento de una anotación intenta tener acceso a un `struct`, `class` o `union` y el miembro con nombre no existe.
