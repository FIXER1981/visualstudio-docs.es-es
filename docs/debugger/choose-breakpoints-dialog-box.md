# <a name="choose-breakpoints-dialog-box"></a>Cuadro de diálogo Elegir puntos de interrupción
El cuadro de diálogo **Elegir puntos de interrupción** se abre al intentar establecer un punto de interrupción a partir de una entrada ambigua en el cuadro de diálogo **Nuevo punto de interrupción**. Un ejemplo común de entrada ambigua es el de un nombre de función sobrecargada.

 **Los puntos de interrupción** se enumeran los posibles puntos de interrupción que se pueden establecer en función de la entrada ambigua. Se puede seleccionar uno o varios puntos de interrupción para establecer.

 **Todos los** selecciona todos los puntos de interrupción aparece en el **puntos de interrupción** cuadro.

 **Ninguno** borra todos los puntos de interrupción aparece en el **puntos de interrupción** cuadro.
