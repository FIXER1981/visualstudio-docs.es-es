# <a name="c28304"></a>C28304
C28304 de ADVERTENCIA: En C++ el caso de Reference-Parameter < parameter_name >, se ha encontrado un operador `_Notref_` colocado incorrectamente en \<token >.

 El operador `_Notref_` solo se debe usar en circunstancias especiales C++ que impliquen parámetros de referencia y solo en macros proporcionadas por el sistema. Debe ir seguido inmediatamente de un operador `_Deref_` o una anotación funcional.
