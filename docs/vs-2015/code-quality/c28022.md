# <a name="c28022"></a>C28022
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

ADVERTENCIA C28022: las clases de función \<classlist1 > en esta función no coinciden con las clases de función \<classlist2 > en la definición de tipo que se usa para definirla.  
  
 Esta advertencia se indica cuando hay un error en las anotaciones. Tanto la definición de tipo como la propia función tienen `_Function_class_` anotaciones, pero no coinciden. Si se usan ambos, deben coincidir.
