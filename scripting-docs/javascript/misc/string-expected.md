# <a name="string-expected"></a>Se esperaba una cadena
Intentó invocar el método **String. prototype. ToString** o **String. prototype.** typeof en un objeto de un tipo distinto de `String`. El objeto de este tipo de invocación debe ser de tipo `String`.  
  
### <a name="to-correct-this-error"></a>Para corregir este error  
  
- Solo se invocan los métodos **String. prototype. ToString** o **String. prototype. valueto** en objetos de tipo `String`.
