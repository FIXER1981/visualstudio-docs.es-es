# <a name="vscodewindowmanager-object"></a>Objeto VSCodeWindowManager

El servicio de lenguaje implementa el administrador de ventanas de código y es responsable de administrar los adornos (por ejemplo, la barra desplegable). Para obtener más información, consulte [Personalización](/visualstudio/extensibility/customizing-code-windows-by-using-the-legacy-api?view=vs-2015)de Windows de código mediante la API heredada .

En la tabla siguiente se `VSCodeWindowManager` muestran las interfaces del objeto.

|Interfaz|Descripción|
|---------------|-----------------|
|<xref:Microsoft.VisualStudio.TextManager.Interop.IVsCodeWindowManager>|Permite agregar o quitar adornos (como barras desplegables) de una ventana de código.|
