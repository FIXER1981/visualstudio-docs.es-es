# <a name="c28226"></a>C28226
Advertencia C28226: La anotación no puede ser un PrimOp (en la declaración actual)

 Esta advertencia indica que la anotación con nombre se declara como un PrimOp y también se declaró previamente como una anotación normal. Esto no debería ser posible si se utilizan las macros estándares para anotaciones. Esta advertencia no se espera que se muestra en el uso normal.
