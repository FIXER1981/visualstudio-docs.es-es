# <a name="imachinedebugmanager-interface"></a>IMachineDebugManager (Interfaz)
La interfaz principal para el administrador de depuración del equipo. Esta interfaz es similar a la interfaz `IMachineDebugManagerCookie`.  
  
 Además de los métodos heredados de `IUnknown`, la interfaz de `IMachineDebugManager` expone los métodos siguientes.  
  
## <a name="methods-in-vtable-order"></a>Métodos en orden de Vtable  
  
|Método|Descripción|  
|------------|-----------------|  
|[IMachineDebugManager::AddApplication](../../winscript/reference/imachinedebugmanager-addapplication.md)|Agrega una aplicación a la lista de aplicaciones en ejecución.|  
|[IMachineDebugManager::RemoveApplication](../../winscript/reference/imachinedebugmanager-removeapplication.md)|Quita una aplicación de la lista de aplicaciones en ejecución.|  
|[IMachineDebugManager::EnumApplications](../../winscript/reference/imachinedebugmanager-enumapplications.md)|Devuelve un enumerador de la lista actual de aplicaciones en ejecución.|
