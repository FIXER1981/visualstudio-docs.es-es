# <a name="c28232"></a>C28232
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

ADVERTENCIA C28232: \_ pre \_ , \_ post \_ o \_ deref \_ no se aplican a ninguna anotación  
  
 Esta advertencia indica que un `_Pre_` `_Post_` operador, o `_Deref_` aparece en una expresión de anotación sin una anotación funcional subsiguiente; se omitió el modificador, pero esto indica una anotación codificada incorrectamente.
