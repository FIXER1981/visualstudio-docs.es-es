# <a name="c28246"></a>C28246
ADVERTENCIA C28246: la anotación de la función ' \<name > ' \<parameter > ' no coincide con el tipo del parámetro

 Se aplicó un operador __deref a un tipo que no es de puntero al crear una anotación.
