# <a name="c28195"></a>C28195
C28195 de ADVERTENCIA: La función se declaró como la adquisición de memoria en una variable y se cerró sin hacerlo.

 Esta advertencia indica que el prototipo de función de la función que se está analizando tiene una anotación `__drv_acquiresMemory`. La anotación `__drv_acquiresMemory` indica que la función adquiere memoria en la ubicación de resultados designada, pero en al menos una ruta de acceso, la función no adquirió la memoria. Tenga en cuenta que la herramienta de análisis de código no reconocerá la implementación real de un asignador de memoria (que implique la aritmética de direcciones) y no reconocerá que la memoria está asignada (aunque se reconozcan muchos contenedores). En este caso, la herramienta de análisis de código no reconoce que la memoria se asignó y emite esta advertencia. Para suprimir el falso positivo, use una advertencia `#pragma` en la línea que precede a la llave de apertura `{` del cuerpo de la función.
