# <a name="c28022"></a>C28022
C28022 de ADVERTENCIA: Las clases de función \<classlist1 > en esta función no coinciden con las clases de función \<classlist2 > en la definición de tipo que se usa para definirla.

 Esta advertencia se indica cuando hay un error en las anotaciones. Tanto la definición de tipo como la propia función tienen anotaciones `_Function_class_`, pero no coinciden. Si se usan ambos, deben coincidir.
