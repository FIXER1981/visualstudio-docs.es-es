# <a name="idiasessionfindacceleratorinlineesbyname"></a>IDiaSession::findAcceleratorInlineesByName
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Devuelve una enumeración de símbolos para los marcos insertados correspondientes al nombre de función insertada especificado.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp#  
HRESULT findAcceleratorInlineeLinesByName (   
   LPCOLESTR             name,  
   DWORD                 option,  
   IDiaEnumSymbols**     ppResult  
);  
```  
  
#### <a name="parameters"></a>Parámetros  
 `name`  
 de Nombre de la función insertada que se va a buscar.  
  
 `option`  
 de Opciones de búsqueda de nombres que se van a usar al buscar fotogramas insertados que se corresponden con `name` . Para obtener más información, vea [enumeración namesearchoptions (](../../debugger/debug-interface-access/namesearchoptions.md).  
  
 `ppResult`  
 enuncia Un puntero a un `IDiaEnumSymbols` puntero de interfaz que se inicializa con el resultado.  
  
## <a name="return-value"></a>Valor devuelto  
 Si la operación se realiza correctamente, devuelve `S_OK`; de lo contrario, devuelve un código de error.  
  
## <a name="remarks"></a>Comentarios  
 Esta función busca Inlines solo dentro de las funciones de código auxiliar de acelerador. Omite los registros de procedimientos nativos de C++.
