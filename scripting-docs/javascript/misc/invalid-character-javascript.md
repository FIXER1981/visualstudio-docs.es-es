# <a name="invalid-character-javascript"></a>Carácter no válido (JavaScript)
Ha compuesto un identificador con un carácter (o caracteres) que el compilador [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] no reconoce como válido. Los caracteres válidos usan las siguientes reglas:  
  
- El primer carácter debe ser una letra ASCII (en mayúsculas o minúsculas) o un carácter de subrayado (_).  
  
- Los caracteres siguientes pueden ser letras ASCII, números o caracteres de subrayado.  
  
- El nombre del identificador no puede ser una palabra reservada.  
  
### <a name="to-correct-this-error"></a>Para corregir este error  
  
- Evite el uso de caracteres que no formen parte de la definición del lenguaje [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)].
