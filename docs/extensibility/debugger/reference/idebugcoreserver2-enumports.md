# <a name="idebugcoreserver2enumports"></a>IDebugCoreServer2::EnumPorts
Recupera una lista de todos los puertos disponibles.

## <a name="syntax"></a>Sintaxis

```cpp
HRESULT EnumPorts( 
   IEnumDebugPorts2** ppEnum
);
```

```csharp
int EnumPorts( 
   out IEnumDebugPorts2 ppEnum
);
```

## <a name="parameters"></a>Parámetros
`ppEnum`\
[fuera] Devuelve un [objeto IEnumDebugPorts2](../../../extensibility/debugger/reference/ienumdebugports2.md) que contiene una lista de todos los puertos de todos los proveedores de puertos.

## <a name="return-value"></a>Valor devuelto
 Si la operación se realiza correctamente, devuelve `S_OK`; de lo contrario, devuelve un código de error.
