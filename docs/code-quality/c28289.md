# <a name="c28289"></a>C28289
C28289 de ADVERTENCIA: Para la función: Readto o grabable no tenía una especificación de límite como parámetro

 La herramienta de análisis de código notifica esta advertencia cuando la anotación de función/parámetro está mal codificada como se indica.
