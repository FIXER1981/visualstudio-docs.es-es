# <a name="c28224"></a>C28224
ADVERTENCIA C28224: la anotación requiere parámetros

 Esta advertencia indica que la anotación con nombre se usa sin parámetros y se requiere al menos un parámetro. Esto no debería ser posible si las macros estándar se utilizan para las anotaciones. no se espera que esta advertencia se vea en el uso normal.
