# <a name="c6272"></a>C6272
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

ADVERTENCIA C6272: no flotante pasado como argumento \<number> cuando se requiere Float en la llamada a \<function>  
  
 Esta advertencia indica que la cadena de formato especifica que se requiere un valor flotante, por ejemplo, una `%f` `%g` especificación de o para pero que no es Float, como `printf,` un entero o una cadena que se está pasando. Es probable que este defecto genere resultados incorrectos. sin embargo, en determinadas circunstancias podría producirse un bloqueo.  
  
## <a name="example"></a>Ejemplo  
 El código siguiente genera esta advertencia:  
  
```  
#include <stdio.h>  
#include <string.h>  
  
void f()  
{  
   char buff[5];  
   int i=5;  
  
   sprintf(buff,"%s %f","a",i);   
}  
  
```  
  
 Para corregir esta advertencia, use en `%i` lugar de `%f` Specification, tal y como se muestra en el código siguiente:  
  
```  
#include <stdio.h>  
#include <string.h>  
  
void f()  
{  
   char buff[5];  
   int i=5;  
  
   sprintf(buff,"%s %i","a",i);    
}  
```  
  
 En el código siguiente se usa la función de manipulación de cadenas seguras, `sprintf_s` , para corregir esta ADVERTENCIA:  
  
```  
#include <stdio.h>  
#include <string.h>  
  
void f()  
{  
   char buff[5];  
   int i=5;  
  
   sprintf_s(buff,5,"%s %i","a",i); // safe version  
}  
```
