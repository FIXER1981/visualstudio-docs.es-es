# <a name="collection-activity-designers"></a>Diseñadores de actividades de colecciones
Los diseñadores de actividades de colecciones se utilizan para crear y configurar las actividades que se utilizan para administrar colecciones. Los temas que se incluyen en esta sección y que aparecen en la siguiente tabla ofrecen orientación sobre cómo utilizar la actividad [!INCLUDE[wfd2](../includes/wfd2-md.md)] y diseñadores de plantillas.

## <a name="in-this-section"></a>En esta sección
 [@No__t_1T AddToCollection >](../workflow-designer/addtocollection-t-activity-designer.md) Se utiliza para crear y configurar una actividad de <xref:System.Activities.Statements.AddToCollection%601>.

 [@No__t_1T ClearCollection >](../workflow-designer/clearcollection-t-activity-designer.md) Se utiliza para crear y configurar una actividad de <xref:System.Activities.Statements.ClearCollection%601>.

 [@No__t_1T ExistsInCollection >](../workflow-designer/existsincollection-t-activity-designer.md) Se utiliza para crear y configurar una actividad de <xref:System.Activities.Statements.ExistsInCollection%601>.

 [@No__t_1T RemoveFromCollection >](../workflow-designer/removefromcollection-t-activity-designer.md) Se utiliza para crear y configurar una actividad de <xref:System.Activities.Statements.RemoveFromCollection%601>.
