# <a name="ca0502"></a>CA0502

Lenguaje no admitido.

Este error se produce cuando el usuario intenta analizar un proyecto de ASP.NET que contiene una o varias páginas o clases escritas en un lenguaje C# distinto de o Visual Basic. Para corregir este error, quite el archivo del proyecto o deshabilite el análisis de código.
