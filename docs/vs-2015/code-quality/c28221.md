# <a name="c28221"></a>C28221
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Advertencia C28221: Se esperaba una expresión de cadena para el parámetro de la anotación  
  
 Esta advertencia indica que se espera un parámetro a una anotación sea una cadena, y se encontró algún otro tipo. Normalmente, esto indica una macro de anotación de código incorrecto y no se espera que se muestra en el uso normal.
