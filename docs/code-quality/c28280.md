# <a name="c28280"></a>C28280
ADVERTENCIA C28280: para el símbolo, se encontró un ' End ' sin el ' begin ' correspondiente

 El lenguaje de anotación admite una construcción de inicio y final (`{` y `}` en C), y el emparejamiento ha ido desequilibrado. Esta situación se puede evitar si se usan las macros.
