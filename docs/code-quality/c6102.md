# <a name="c6102"></a>C6102
C6102 de ADVERTENCIA: Usar \<variable > de una llamada de función errónea en la línea \<Location >.

 Esta advertencia se indica en lugar de [C6001](../code-quality/c6001.md) cuando no se establece una variable porque se marcó como un parámetro `_Out_` en una llamada de función anterior en la que se produjo un error.

 El problema puede ser que la función llamada anteriormente no está anotada completamente. Puede requerir `_Always_`, `_Outptr_result_nullonfailure_` (`_COM_Outptr_` para código COM) o una anotación relacionada.
